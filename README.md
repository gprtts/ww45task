\# Параллельный расчёт суммы квадратов (Task)



\## 📌 Описание



Программа вычисляет сумму квадратов массива чисел с использованием параллельных задач (`Task`).



Каждое число обрабатывается в отдельной задаче, что позволяет ускорить вычисления.



\## ⚙️ Используемые технологии



\* C#

\* .NET

\* Task (многопоточность)



\## 🚀 Как работает



1\. Создаётся массив чисел

2\. Для каждого числа запускается `Task`

3\. Считается квадрат числа

4\. Результаты суммируются с использованием `lock`

5\. Выводятся:



&#x20;  \* квадрат числа

&#x20;  \* ID потока

&#x20;  \* итоговая сумма



\## 📊 Пример вывода



Число: 2, квадрат: 4, поток: 5

Число: 3, квадрат: 9, поток: 7

Итоговая сумма квадратов: 55



\## ❗ Особенности



\* Используется `lock` для предотвращения race condition

\* Используется `Task.WaitAll()` для синхронизации
